## An AI-powered customer support assistant built with RAG, conversation memory, and OpenAI.

# Features

- Answers customer questions from company FAQ data
- Uses semantic search with embeddings
- Supports follow-up questions with conversation memory
- Shows sources used
- Refuses to answer when information is unavailable

# Tech Stack

- Python
- Google Colab
- OpenAI API
- LangChain
- ChromaDB

# Project Status

Prototype completed in Google Colab. FastAPI deployment coming next.
